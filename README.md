# 🚀 Job Portal System

A full-stack **Job Portal Application** built using **Spring Boot (Backend)** and **React + Tailwind CSS (Frontend)**.
Designed with clean architecture and scalable backend practices.

---

## 🧠 Overview

This system provides:

* Secure user authentication 🔐
* User management 👤
* Job management 💼

---

## 🏗️ Tech Stack

### Backend

* Java 17
* Spring Boot
* Spring Security
* Spring Data JPA
* PostgreSQL
* Maven
* Lombok

### Frontend

* React.js
* Tailwind CSS

---

## 📂 Project Structure

```
server/job-portal
│
├── controller
├── service
│   └── impl
├── repository
├── entity
├── dto
├── config
├── exception
```

---

## ⚙️ Setup

1. Clone the repository
2. Create PostgreSQL database: `job_portal`
3. Configure `application.yaml`
4. Run the application

---

## 📌 Highlights

* Layered architecture (Controller → Service → Repository)
* DTO-based validation
* Exception handling
* Scalable backend design
