# Job Portal Backend

This is the backend of the Job Portal project built using Spring Boot.

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- MySQL

## How to Run

1. Go to server folder:
   cd server/job-portal

2. Run project:
   mvn spring-boot:run

## APIs
- POST /api/auth/register
- POST /api/auth/login
